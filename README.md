# 🌌 MICROTONAL_DAW

> Минималистичная ретро-геометрическая рабочая станция для исследования, прослушивания и создания многодорожечной музыки в динамических микротональных темперациях (EDO).

---

## 💾 Сохранение, Импорт и Структура Проектов (JSON)

DAW полностью автономна и позволяет экспортировать ваши песни и настройки в файлы формата `.json` для последующей загрузки или обмена с другими музыкантами.

В корне данного репозитория лежит готовый пример трека — **`asteria.json`** (16 тактов с импровизированными ударными, пэдом, басом и мелодией). Скачайте его и нажмите кнопку **LOAD** на сайте, чтобы мгновенно прослушать!

### 📊 Спецификация формата данных (JSON Schema)

Каждый файл проекта представляет собой структурированный объект со следующими ключевыми полями:

```json
{
  "edo": 31,                // Текущая темперация октавы (от 5 до 72)
  "tempo": 130,             // Темп композиции в BPM (от 60 до 220)
  "currentScale": "major",   // Активная маска лада для сот и круга
  "showCircleLabels": true, // Видимость цифр над активными нотами круга
  
  "tracks": [               // Список физических дорожек на таймлайне
    { "id": "track_1", "name": "LEAD MELODY" }
  ],
  
  "blocks": [               // Массив музыкальных кадров (клипов)
    {
      "id": "block_unique_id",
      "trackId": "track_1",      // К какому потоку привязан клип
      "startBeat": 0.0,          // Время начала проигрывания (в долях)
      "durationBeats": 4.0,      // Длина клипа (в долях, квантование по 1/32)
      "instrumentId": "saw",     // Какой синт проигрывает этот кадр
      "baseFreq": 261.63,        // Индивидуальная тональность кадра в Гц
      "velocity": 100,           // Сила нажатия / Громкость кадра (0 - 127)
      "notes": [0, 10, 18]       // Шаги EDO, звучащие одновременно (аккорд)
    }
  ],
  
  "instruments": [          // Индивидуальные настройки каждого из синтов
    {
      "id": "saw",
      "name": "SAW",
      "color": "#ED6ED8",        // Фирменный неоновый цвет
      "waveType": "sawtooth",    // Форма волны (sine, triangle, sawtooth, square)
      "attack": 100,             // Атака огибающей (в миллисекундах)
      "decay": 300,              // Спад огибающей
      "sustain": 90,             // Удержание огибающей (MIDI 0 - 127)
      "release": 1500,           // Затухание огибающей (в миллисекундах)
      "reverb": 0.3,             // Степень пространственной реверберации (0.0 - 1.0)
      "delay": 0.1,              // Степень эха задержки (0.0 - 1.0)
      "a_disabled": false        // Флаги ручного отключения фаз ADSR
    }
  ]
}
```

## 🚀 Основные возможности (Features)

*   **Динамический EDO-строй:** Свободно делите октаву на любое количество равных частей (от 5 до 72 EDO) прямо во время игры.
*   **11-орбитальный концентрический радар-визуализатор:** Уникальное круговое поле координат (от октавы 0 до октавы 10). Аккорды строятся как объемные многоугольники, натягивающие неоновые грани между орбитами.
*   **Адаптивная изоморфная клавиатура сот:** Гексагональная сетка (соты), расширенная до 3 полных октав с возможностью мгновенного сдвига (Octave Shift) от баса C0 до ультра-верхов B10. Лишние ноты автоматически скрываются из сетки по краям диапазона.
*   **Универсальный Scale-фильтр:** Мгновенное наложение масок ладов (Chromatic, Major, Minor, Pentatonic, Just Intonation 5-limit) на соты и круг, блокирующее негармоничные ноты.
*   **Pre-Fader мастер-анализатор:** Живой осциллограф и фигуры Лиссажу под кругом всегда показывают упругую пиксельную форму волны, даже если вы убавили мастер-громкость в ноль.
*   **Мульти-трековый параллельный таймлайн:** Компонуйте музыку из параллельно звучащих кадров разных инструментов. Блоки можно плавно перетаскивать мышкой между дорожками и растягивать за правый край с точностью до 1/32 доли.
*   **Редактор звука и ADSR:** Тонкая прямоугольная панель ADSR с нативными вертикальными фейдерами и точным числовым вводом для каждого кадра, дополненная пространственными эффектами Reverb и Delay.
*   **Аппаратный скраббинг и Panic-кнопка:** Плавное перемещение бегунка по линейке тактов с привязкой по 1/32. Двойной клик на STOP мгновенно гасит все синтезаторы и звуковые хвосты эффектов.

## 🎧 Инструменты и Цветовая гамма (Deltarune-palette)

*   `TRIANGLE` — Зеленый (`#59DC90`): мягкий, певучий флейтовый тембр.
*   `SAW` — Розовый (`#ED6ED8`): агрессивная, богатая гармониками пила.
*   `SQUARE` — Голубой (`#7FFDEB`): классический 8-битный ретро-звук.
*   `SINE` — Желтый (`#FFCE32`): чистый, глубокий саб-бас и бочка.

---

## ⌨️ Горячие клавиши (Hotkeys)

*   `Space` (Пробел) — Запуск / Пауза воспроизведения.
*   `Esc` — Снять фокус (деселект) с выделенного кадра.
*   `Ctrl + D` (или `Cmd + D`) — Продублировать выделенный кадр встык.
*   **Двойной клик на STOP (■):** Полный аудио-сброс (Panic).
*   **Буквенные ряды клавиатуры ПК:** Полноценное микротональное пианино с поддержкой записи в кадр.

## 🛠 Установка и запуск локально (Local Setup)

Для запуска проекта на своем компьютере у вас должен быть установлен Node.js.

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/stornov/microtonal-daw.git

# 2. Перейдите в папку проекта
cd microtonal-daw

# 3. Установите зависимости
npm install

# 4. Запустите сервер разработки
npm run dev
```

Откройте `http://localhost:5173` в браузере для работы с приложением.
